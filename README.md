# B.I.C. Monorepo

В репозитории лежит приложение `marketplace` — Next.js‑витрина с админ‑панелью и Postgres/Drizzle.  
Ниже краткий план запуска проекта на своей машине; за подробностями загляните в `marketplace/README.md`.

## Быстрый старт

1. **Клонирование**
   ```ps1
   git clone <URL> bic
   cd bic
   ```
2. **Перейдите в директорию приложения**
   ```ps1
   cd marketplace
   ```
3. **Следуйте инструкции из `marketplace/README.md`**  
   Скомандует: создать `.env.local`, запустить Postgres (Docker), сделать `npm install`, миграции и сиды, после чего `npm run dev`.

## Структура

- `marketplace/` — исходники Next.js приложения, скрипты миграций и сидов, документация по запуску.
- `docs/` (если используется) — дополнительные материалы и макеты.

Дополнительно: при появлении новых пакетов/сервисов кладите поддиректории рядом и дополняйте этот README ссылками на их документацию.
